## 三、详细问题分析

### 3.1 API 调用配置问题

**当前状态**：
```typescript
// src/utils/api.ts
const API_BASE = import.meta.env.VITE_API_URL || '';  // 空字符串！
```

**问题**：
- `.env.example` 中 `VITE_API_URL=` 为空
- 部署时必须设置 `VITE_API_URL=https://your-hf-space-name.hf.space`
- 否则所有请求发往同源（Cloudflare Pages），返回 404

**解决方案**：
```bash
# 在 Cloudflare Pages 环境变量中设置
VITE_API_URL=https://your-hf-space-name.hf.space
```

---

### 3.2 模型加载问题（HuggingFace Spaces）

**当前代码** (`app.py:22-24`)：
```python
# 硬编码本地路径，HF Spaces 上不存在
model_roberta = AutoModelForSequenceClassification.from_pretrained("./models/roberta-compliance")
model_mt5 = MT5ForConditionalGeneration.from_pretrained("./models/mt5-compliance")
```

**问题**：
- HF Spaces 没有持久化存储 `/models` 目录
- 每次 Space 重启模型文件丢失
- 需要使用 HF Hub 或嵌入模型文件

**解决方案**：
```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification, AutoModelForSeq2SeqLM

# 使用 HuggingFace 预训练模型
model_roberta = AutoModelForSequenceClassification.from_pretrained(
    "hfl/chinese-roberta-wwm-ext",
    num_labels=12
)
tokenizer_roberta = AutoTokenizer.from_pretrained("hfl/chinese-roberta-wwm-ext")

model_mt5 = MT5ForConditionalGeneration.from_pretrained("google/mt5-base")
tokenizer_mt5 = AutoTokenizer.from_pretrained("google/mt5-base")
```

### 3.4 文件上传限制

**当前代码**：
```python
@app.post("/api/v1/upload")
async def upload_file(file: UploadFile = File(...)):
    content = await file.read()
    text = content.decode("utf-8", errors="ignore")
```

**问题**：
- 无文件大小限制
- 只支持纯文本，PDF/Word 需额外处理

**建议实现**：
```python
MAX_FILE_SIZE = 5 * 1024 * 1024  # 5MB

@app.post("/api/v1/upload")
async def upload_file(file: UploadFile = File(...)):
    content = await file.read()
    
    if len(content) > MAX_FILE_SIZE:
        raise HTTPException(status_code=413, detail="文件大小超过5MB限制")
    
    # 检测文件类型
    if file.filename.endswith('.pdf'):
        text = extract_pdf(content)
    else:
        text = content.decode("utf-8", errors="ignore")
    
    return {"text": text}
```
