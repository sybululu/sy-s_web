## Task 4: 重新部署 HF Space（需要用户操作）

**用户需要手动执行以下步骤：**

- [ ] **Step 1: 删除并重新创建 HF Space**

1. 访问 https://huggingface.co/new-space (选择创建新Space)
2. 选择 SDK: "Gradio"
3. Space name: `privacy-policy-checker`
4. License: 选择合适许可
5. 点击 "Create Space"

- [ ] **Step 2: 上传代码到新创建的 Space**

可以使用以下任一方式：

**方式A: 使用 Git**
```bash
git clone https://huggingface.co/spaces/[your-username]/privacy-policy-checker
# 复制 app.py, analyzer.py, requirements.txt 到克隆目录
cd privacy-policy-checker
git add .
git commit -m "Initial commit"
git push
```

**方式B: 使用网页上传**
在 HF Space 页面直接拖拽上传文件：
- `app.py`
- `analyzer.py`
- `requirements.txt`
- `models.py`
